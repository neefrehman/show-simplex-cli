# show-simplex

A CLI that displays an animated ASCII [simplex noise](https://en.wikipedia.org/wiki/Simplex_noise) field in your terminal. You may have to squint a bit.

### Installation

```shell
npm install show-simplex -g
```

### Usage

```
show-simplex
```
